# 一.如何注册 BitMart
## 1. 使用APP端 （无需VPN)

* 步骤一：打开APP，选择注册，进入注册页面
* 步骤二：输入【邮箱】【密码】，勾选用户协议，点击【注册】
* 步骤三：进行图形验证，点击【确定】
* 步骤四：进行邮箱验证码验证，输入【6位数验证码】点击下一步
* 步骤五：提示【注册成功】表示您已经注册成功

![](./images/zh_register_app.png)


## 2. 使用手机浏览器 （无需VPN)

* 步骤一：打开手机的浏览器，输入网址 https://www.bm-link.com/register/zh-CN, 进入注册页面
* 步骤二：输入【邮箱】【密码】，勾选用户协议，点击【注册】
* 步骤三：进行图形验证，点击【确定】
* 步骤四：进行邮箱验证码验证，输入【6位数验证码】点击下一步
* 步骤五：提示【注册成功】表示您已经注册成功
